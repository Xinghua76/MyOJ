## 🔧 常见问题

### Q: 为什么 AI 按钮是灰色的？
**A:** 代码编辑区为空。请先编写至少一行代码，按钮会自动启用。

### Q: 复制了什么内容？
**A:** 纯文本内容（已移除 Markdown 格式标记），可以直接粘贴到其他应用。

### Q: 能否不关闭抽屉直接重新分析？
**A:** 可以！点击"重新分析"按钮，无需关闭，直接重新调用 API。

### Q: 为什么有时显示"系统预设建议"？
**A:** API Key 无效或 AI 服务不可用时，系统会自动返回预设建议（仍然有价值）。

### Q: 复制功能为什么有时不工作？
**A:** 
- 浏览器安全限制：使用了降级方案（textarea.select）
- 某些浏览器隐私模式可能不支持
- 尝试手动复制或刷新页面后重试

## 📝 调试步骤

如果某个功能不工作：

1. **打开浏览器控制台**
   ```
   F12 或 右键 → 检查元素 → Console
   ```

2. **检查是否有错误信息**
   ```
   看是否有红色错误日志
   ```

3. **检查网络请求**
   ```
   Network 选项卡 → 点击 AI 分析
   → 查看 /api/ai/analyze 请求是否成功
   ```

4. **重新加载页面**
   ```
   Ctrl+Shift+R 或 Cmd+Shift+R（硬刷新）
   ```
